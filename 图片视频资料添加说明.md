# 精工智造网站图片和视频资料添加说明

## 1. 概述

本文档详细说明如何为精工智造网站添加图片和视频资料，以便更好地展示您的CNC立式加工中心服务和作品。

## 2. 图片资料添加

### 2.1 图片存储位置

所有图片资料应存储在以下目录中：
- 作品展示图片：`precision_machining_website/media/works/images/`
- 其他静态图片：`precision_machining_website/static/images/`

### 2.2 图片格式要求

为了保证网站质量和加载速度，请遵循以下图片格式要求：
- 格式：JPEG、PNG 或 GIF
- 推荐尺寸：
  - 作品展示图片：800x600 像素或更大比例相似
  - 缩略图：300x200 像素
- 文件大小：单张图片不超过 2MB
- 命名规范：使用英文命名，如 `cnc_machine_part.jpg`

### 2.3 添加图片到作品

在 [populate_data.py](file://d:/Users/00596/Desktop/%E7%B2%BE%E5%B7%A5%E5%88%9B%E5%AE%A2/precision_machining_website/populate_data.py) 文件中，可以通过 `image` 字段指定图片路径：

```python
{
    'title': '示例作品',
    'description': '作品描述',
    'category': categories[0],
    'image': 'works/images/cnc_machine_part.jpg',  # 图片路径
    # 其他字段...
}
```

### 2.4 批量上传图片

1. 将准备好的图片复制到对应目录
2. 确保图片命名符合规范
3. 在 [populate_data.py](file://d:/Users/00596/Desktop/%E7%B2%BE%E5%B7%A5%E5%88%9B%E5%AE%A2/precision_machining_website/populate_data.py) 中更新作品数据，指定正确的图片路径
4. 运行数据填充脚本更新数据库：
   ```bash
   python populate_data.py
   ```

## 3. 视频资料添加

### 3.1 视频来源

网站支持两种视频嵌入方式：
1. YouTube 视频链接
2. 其他在线视频平台的嵌入链接

### 3.2 添加视频到作品

在 [populate_data.py](file://d:/Users/00596/Desktop/%E7%B2%BE%E5%B7%A5%E5%88%9B%E5%AE%A2/precision_machining_website/populate_data.py) 文件中，通过 `video_url` 字段添加视频链接：

```python
{
    'title': '示例作品',
    'description': '作品描述',
    'category': categories[0],
    'video_url': 'https://www.youtube.com/embed/video_id',  # 视频链接
    # 其他字段...
}
```

### 3.3 视频格式要求

- 推荐使用 YouTube 或其他主流视频平台托管视频
- 视频应展示实际加工过程、成品展示或设备演示
- 视频长度建议控制在 1-3 分钟内

## 4. 最佳实践建议

### 4.1 图片选择建议

1. 选择光线充足、焦点清晰的照片
2. 展示加工过程的关键环节
3. 包含成品的多角度视图
4. 展示加工设备和工作环境
5. 对比图（加工前后对比）

### 4.2 视频内容建议

1. 设备加工过程演示
2. 成品质量检测过程
3. 工厂环境和生产能力展示
4. 技术人员专业操作展示
5. 客户反馈和评价

### 4.3 版权注意事项

1. 仅使用自己拥有版权或授权使用的图片和视频
2. 如使用第三方素材，请确保获得合法授权
3. 避免在素材中展示其他公司的商标或敏感信息

## 5. 更新流程

1. 准备并上传新的图片和视频资料
2. 修改 [populate_data.py](file://d:/Users/00596/Desktop/%E7%B2%BE%E5%B7%A5%E5%88%9B%E5%AE%A2/precision_machining_website/populate_data.py) 文件中的作品数据
3. 运行以下命令更新网站内容：
   ```bash
   python manage.py migrate  # 如果有模型变更
   python populate_data.py   # 更新数据
   ```
4. 启动开发服务器检查效果：
   ```bash
   python manage.py runserver
   ```

## 6. 故障排除

### 6.1 图片无法显示

- 检查图片路径是否正确
- 确认图片文件确实存在于指定位置
- 检查文件权限设置
- 确认图片格式符合要求

### 6.2 视频无法播放

- 检查视频链接是否正确
- 确认视频链接是嵌入式链接而非观看页面链接
- 测试视频链接在浏览器中是否可以直接访问
- 检查网络连接是否正常
