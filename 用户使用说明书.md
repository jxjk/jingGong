# 精工创客网站内容管理系统使用说明书

## 1. 系统概述

精工创客网站内容管理系统是一个基于Django框架构建的专业制造行业展示平台。系统支持分类管理、作品展示、图片浏览等功能，旨在为企业提供高质量的产品展示服务。

## 2. 数据填充脚本使用指南

### 2.1 功能说明

`populate_data.py` 脚本用于快速向网站数据库添加初始内容，包括分类和作品数据。

### 2.2 使用步骤

1. **准备工作**
   ```bash
   pip install -r requirements.txt
   python manage.py migrate
   ```

2. **运行数据填充脚本**
   ```bash
   python populate_data.py
   ```

3. **查看结果**
   ```bash
   python manage.py runserver
   ```
   访问 http://127.0.0.1:8000 查看网站首页

### 2.3 脚本功能详解

脚本会自动执行以下操作：
- 创建4个主要分类：
  - 精密零件 (slug: precise-part)
  - 工装夹具 (slug: fixture)
  - 生产线集成 (slug: production-line)
  - 概念设计 (slug: concept-design)
- 在每个分类下创建3个示例作品
- 为每个作品设置标题、描述、图片路径和重要性等级(1-5)

## 3. 管理后台使用指南

### 3.1 访问管理后台

网站管理后台是Django内置的管理界面，允许用户方便地管理网站内容。要访问管理后台：

1. 启动开发服务器：
   ```bash
   python manage.py runserver
   ```

2. 在浏览器中访问：http://127.0.0.1:8000/admin/

3. 使用管理员账户登录（默认账户在entrypoint.sh中创建）：
   - 用户名：admin
   - 密码：admin

### 3.2 管理内容

管理后台提供以下功能：

#### 3.2.1 分类管理
- 添加新分类：点击"作品分类"旁的"增加"按钮
- 编辑分类：点击分类名称进行编辑
- 删除分类：勾选分类前的复选框，选择"删除"操作

#### 3.2.2 作品管理
- 添加新作品：点击"作品"旁的"增加"按钮
- 编辑作品：点击作品标题进行编辑
- 删除作品：勾选作品前的复选框，选择"删除"操作

### 3.3 管理后台特性

管理后台具有以下特性：
- 直观的用户界面，易于操作
- 支持图片上传和管理
- 可以方便地设置作品的分类和属性
- 支持搜索和筛选功能
- 提供详细的作品信息编辑界面

## 4. 内容自定义

### 4.1 修改分类
编辑 `populate_data.py` 中的 `categories` 列表：
```python
categories = [
    {'name': '分类名称', 'slug': '英文标识'},
    # ...更多分类
]
```

### 4.2 修改作品
编辑各分类下的 `works` 列表：
```python
'works': [
    {
        'title': '作品标题',
        'description': '详细描述',
        'image': '图片路径',
        'importance': 重要性等级(1-5)
    },
    # ...更多作品
]
```

### 4.3 添加新内容
按照现有格式在相应位置添加新的分类或作品条目。

## 5. 网站功能说明

### 5.1 分类浏览
- 首页展示所有分类及其代表性作品
- 点击分类图标进入该分类下的作品列表

### 5.2 作品展示
- 按重要性排序显示作品
- 点击作品图片查看详细信息
- 通过面包屑导航可返回上级页面

### 5.3 响应式设计
- 支持PC、平板、手机等多种设备访问
- 自适应布局，保证良好的用户体验

## 6. 注意事项与故障排除

### 6.1 重要提醒
- 运行脚本会清除现有分类和作品数据后再重新创建
- 图片文件需放置在 `static/images/` 目录下
- 请备份重要数据后再运行脚本

### 6.2 常见问题解决

| 问题 | 解决方案 |
|------|----------|
| 数据库迁移错误 | 检查数据库连接配置，确认已安装所需依赖 |
| 图片无法显示 | 确认图片路径正确，文件存在于指定位置 |
| 权限不足 | 使用管理员权限运行命令行工具 |
| 页面404错误 | 检查URL配置和视图函数实现 |
| 管理后台无法访问 | 确认已创建超级用户账户 |

## 7. 维护建议

1. 定期备份数据库和媒体文件
2. 保持Python环境和依赖包更新
3. 监控网站性能，适时优化数据库查询
4. 根据业务发展需要扩展分类和作品内容
