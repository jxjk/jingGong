# 精工智造平台管理员使用说明

本文档介绍了如何使用精工智造平台的管理员功能来管理客户数据和进行客户回访。

## 管理员账户设置

### 1. 创建超级用户
在项目根目录下执行以下命令创建管理员账户：

```bash
cd precision_machining_website
python manage.py createsuperuser
```

按照提示输入用户名、邮箱和密码完成创建。

### 2. 登录管理后台
访问 `http://your-domain/admin/` 使用刚才创建的账户登录Django管理后台。

## 管理客户数据

### 1. 使用Django管理后台
Django自带的管理后台提供了完整的数据管理功能：

1. 登录后可以看到"报价请求"模块
2. 可以查看、编辑、删除客户提交的报价请求
3. 支持按不同字段进行筛选和搜索
4. 可以导出数据为CSV格式

### 2. 使用专用管理面板
访问 `http://your-domain/quote/admin/dashboard/` 进入专用管理面板：

#### 仪表板功能
- 查看总体统计数据（总报价数、已处理、待处理）
- 查看最近7天的报价趋势图
- 按加工类型统计的饼图
- 快速访问常用功能

#### 报价列表
- 查看所有客户报价
- 支持按姓名、邮箱、电话搜索
- 查看报价详情
- 标记报价为已处理状态

#### 数据导出
- 支持将所有报价数据导出为CSV格式
- 导出字段包括客户信息、项目详情、3D分析结果等

## 客户数据保护措施

### 1. 访问控制
- 只有具有管理员权限的用户才能访问管理功能
- 使用Django内置的权限系统进行访问控制
- 所有管理操作都需要登录验证

### 2. 数据加密建议
对于敏感信息，建议采取以下措施：
- 对客户电话、邮箱等敏感信息进行加密存储
- 使用HTTPS协议传输数据
- 定期备份数据库

### 3. 日志记录
- 系统会记录管理员的重要操作
- 可追踪数据修改历史

## 客户回访策略

### 1. 定期检查待处理报价
- 每日检查待处理的报价请求
- 及时联系客户进行沟通

### 2. 数据分析
- 利用仪表板统计数据识别热门加工类型
- 分析客户需求趋势，优化服务

### 3. 客户跟进
- 对于高价值或复杂的报价请求，应优先跟进
- 建立客户档案，记录沟通历史

## 常见问题处理

### 1. 无法访问管理面板
- 确认已正确创建超级用户
- 检查URL路径是否正确
- 确认用户具有管理员权限

### 2. 数据导出失败
- 检查服务器存储空间
- 确认文件权限设置正确

### 3. 搜索功能无效
- 确认搜索关键词正确
- 检查数据库连接状态

## 安全建议

1. 定期更改管理员密码
2. 不要在公共计算机上保持登录状态
3. 限制管理员账户数量
4. 定期审查管理员操作日志
5. 使用强密码策略
6. 启用双因素认证（如需要）

通过以上功能和措施，管理员可以有效管理客户数据，同时确保客户隐私得到充分保护。
